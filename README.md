# node-nest-server-demo

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```sh
  npm install
```

## Running the app

```sh
  # development
  npm run start

  # watch mode
  npm run start:dev

  # production mode
  npm run start:prod
```

## Test

```sh
  # unit tests
  npm run test

  # e2e tests
  npm run test:e2e

  # test coverage
  npm run test:cov
```
